# 著作物 / Works

**An Unnamed Resistor** の著作物および補足資料を管理するフォルダです。  
This folder contains works and supplementary materials by **An Unnamed Resistor**.

現時点では記事本体のPDFは非公開です。  
Article PDFs are not currently public.

---

## フォルダ構造 / Folder Structure

各著作物は以下の構造で追加されます。  
Each work will follow this structure:

```
works/
└── {article-slug}/
    ├── README.md                         # 記事概要・リンク / Article info & links
    ├── article_v{X.X}_{YYYYMMDD}.pdf     # 記事本体PDF（公開する場合）
    └── supplementary/                    # 補足資料
        ├── README.md
        └── （補足ファイル群 / supplementary files）
```

`article-slug` は英語小文字・ハイフン区切りの識別子です。  
`article-slug` is a lowercase English identifier with hyphens.

例 / Example: `vtuber-structural-analysis`
