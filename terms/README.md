## ファイル命名規則 / File Naming Convention

```
terms_{言語/lang}_{メジャー}.{マイナー}_{YYYYMMDD}.pdf

例 / Example:
  terms_ja_v1.0_20260508.pdf   ← 日本語版 v1.0
  terms_en_v1.0_20260508.pdf   ← 英語版 v1.0
  terms_ja_v1.1_20261201.pdf   ← 日本語版 v1.1（将来の改訂）
```

**メジャー番号を上げる場合 / Major version bump:**  
構造的な変更・重要な解釈変更・条件の追加・削除  
Structural changes, significant reinterpretation, addition or removal of conditions.

**マイナー番号を上げる場合 / Minor version bump:**  
誤字修正・表現の明確化・例示の追加  
Typo fixes, clarification of wording, addition of examples.
